# Implementation Plan for LAI Scheduler Update

## 1. 기존 코드 분석 및 리팩토링 (Code Analysis and Refactoring)
- **목표:** `Otsuka_LAI_scheduler.py` 코드의 현재 구조, 로직, 사용된 라이브러리를 분석하고 향후 개발을 위해 코드를 모듈화 및 재구성합니다.
- **세부 작업:**
    - 데이터 관리, UI, 비즈니스 로직 분리 가능성 검토
    - 변수 및 함수 네이밍 컨벤션 통일
    - 불필요한 코드 제거 및 주석 추가

## 2. UI/UX 개선 (UI/UX Enhancement)
- **목표:** 보다 직관적이고 미려한 디자인의 사용자 인터페이스를 제공합니다.
- **세부 작업:**
    - **반응형 캘린더:** 창 크기 조절 시 캘린더의 행/열 크기가 비율에 맞게 조절되도록 구현
    - **디자인 개선:** 상용 프로그램 수준의 아이콘, 색상, 폰트 적용
    - **오늘 날짜 강조:** 현재 날짜를 시각적으로 쉽게 인지할 수 있도록 배경색 또는 테두리 색상 변경
    - **로고 및 이미지 추가:**
        - 좌측 상단: Janssen Korea 로고 표시
        - 우측 상단: Invega Sustenna 이미지 표시 (부드러운 그라데이션 처리)

## 3. 예약 관리 기능 (Appointment Management)
- **목표:** 사용자가 예약을 손쉽게 생성, 수정, 삭제하고 상태를 관리할 수 있도록 합니다.
- **세부 작업:**
    - **예약 블록 표시:**
        - 예약 상태: 옅은 주황색 블록
        - 완료 상태: 연두색 블록
    - **드래그 앤 드롭 예약 변경:** 캘린더의 예약 블록을 마우스로 끌어 다른 날짜로 이동 기능 구현
    - **예약 수정 및 완료 처리:**
        - 예약 블록 더블 클릭 시, 기존 내용이 채워진 수정 창 팝업
        - '완료' 버튼 클릭 시, 상태를 '완료'로 변경하고 다음 예약 자동 생성
    - **데이터 동기화:** 이전 예약의 수정/삭제가 다음 예약에 정확히 반영되도록 로직 구현

## 4. 알람 기능 (Alarm Feature)
- **목표:** 사용자에게 예정된 투약 일정을 효과적으로 알립니다.
- **세부 작업:**
    - **기본 팝업 알람:** 프로그램 시작 시 알람 기능 기본 활성화
    - **차등적 알람창:**
        - 오늘: 분홍색 계열
        - 내일: 주황색 계열
        - 모레: 하늘색 계열
    - **알람 강제 테스트 기능 제거:** 기존 기능 삭제
    - **추가 알람 설정:** 1일 전, 2일 전 오전 7시에 추가 알람을 받을 수 있는 옵션 제공

## 5. 외부 서비스 연동 (External Service Integration)
- **목표:** Google Calendar와 연동하여 사용자의 스마트폰에서도 일정을 확인할 수 있게 합니다.
- **세부 작업:**
    - Google 계정 정보 입력 UI 구현
    - Google Calendar API 연동을 통한 양방향 동기화 (생성, 수정, 삭제)

## 6. 문서화 및 배포 (Documentation and Deployment)
- **목표:** 최종 사용자가 프로그램을 쉽게 설치하고 사용할 수 있도록 지원합니다.
- **세부 작업:**
    - **사용자 매뉴얼 작성:** 프로그램의 각 기능에 대한 상세한 설명서 제작
    - **실행 파일 생성:** PyInstaller 등을 사용하여 Windows 환경용 `.exe` 파일 생성
