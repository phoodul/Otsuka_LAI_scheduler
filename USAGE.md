# 사용 설명서 (Usage Guide)

## 1. 기본 사용법

### 신규 처방 입력
- 날짜 칸을 **더블 클릭**하거나 **F1** 키를 눌러 신규 처방을 입력합니다.
- 환자명, 약품명, 용량, 간격(일), 메모를 입력 후 **[등록/수정]** 버튼을 누릅니다.
- 저장 시 주사 기록(파란 블록)과 다음 예약(주황 블록)이 자동으로 생성됩니다.

### 예약 완료 처리 (당일 방문)
- 달력에 주황색으로 표시된 **[예정]** 블록을 더블 클릭합니다.
- 내용 확인 또는 수정 후 **[등록/수정]** 버튼을 누르면 해당 날짜에 주사 기록이 생성되고, 다음 예약이 자동으로 계산됩니다.

### 처방 기록 수정
- 달력에 녹색으로 표시된 **[주사]** 블록을 더블 클릭합니다.
- 내용(약품, 용량, 간격 등)을 수정 후 저장하면 연결된 예약일도 함께 업데이트됩니다.
- 예약이 삭제된 상태에서 주사 기록을 수정하면 새로운 예약이 자동으로 재생성됩니다.

---

## 2. 예약일 드래그로 변경

환자가 예약일보다 일찍 또는 늦게 방문한 경우, 예약 블록을 드래그하여 날짜를 수정할 수 있습니다.

1. 주황색 **[예정]** 블록 위에서 **Ctrl + 마우스 클릭**을 유지합니다.
2. 원하는 날짜 칸으로 **드래그**합니다. 이동 가능한 날짜는 파란색으로 하이라이트됩니다.
3. 마우스를 **놓으면** 예약이 해당 날짜로 이동하고, 이전 주사 기록의 간격도 실제 날짜 차이로 자동 업데이트됩니다.
4. 이동된 날짜에서 블록을 **더블 클릭**하여 처방을 완료하면, 다음 예약이 수정된 간격 기준으로 생성됩니다.

> **예시:** 2/20 주사(28일 간격) → 3/20 예약 → 환자가 3/18 방문
> 3/20 블록을 Ctrl+드래그하여 3/18로 이동 → 이전 주사 간격이 28일→26일로 자동 수정
> 3/18 블록 더블클릭 후 등록 → 다음 예약이 3/18+26일 = 4/13으로 생성

## 3. 기록 삭제

날짜 칸을 더블 클릭하면 하단에 해당 날짜의 기록 목록이 표시되고, 각 항목 오른쪽의 **[삭제]** 버튼으로 삭제할 수 있습니다.

| 삭제 대상 | 동작 |
|-----------|------|
| **[주사] 기록** | 주사 기록 + 연결된 다음 예약 함께 삭제 |
| **[예정] 예약** | 예약만 삭제, 이전 주사 기록은 유지 |

## 4. 알림 기능

프로그램 시작 시 및 매일 오전 7시에 예정 환자 알림 팝업이 자동으로 표시됩니다.

- **오늘(D-Day):** 분홍색 팝업 — 오늘 주사 예약 환자
- **내일(D+1):** 주황색 팝업 — 내일 예약 환자
- **모레(D+2):** 파란색 팝업 — 모레 예약 환자

하단의 **내일 알림 / 모레 알림** 체크박스로 각 알림을 켜고 끌 수 있습니다.

### 알림 창 단축키

| 단축키 | 동작 |
|--------|------|
| `Alt + K` | 모레 → 내일 → 오늘 순서로 팝업 하나씩 닫기 |
| `Shift + Alt + K` | 열린 알림 팝업 모두 한 번에 닫기 |

## 6. 스마트폰 캘린더 연동 (iCalendar 동기화)

이 프로그램은 Google Drive나 OneDrive 같은 클라우드 스토리지의 데스크톱 동기화 기능을 이용하여 `schedule.ics` 캘린더 파일을 자동으로 동기화할 수 있습니다.

### 초기 설정 (최초 1회)
1. **클라우드 데스크톱 앱 설치:** 사용하시는 컴퓨터에 [Google Drive 데스크톱](https://www.google.com/drive/download/) 또는 [Microsoft OneDrive](https://www.microsoft.com/ko-kr/microsoft-365/onedrive/download) 앱이 설치되어 있고, 로그인 되어 있는지 확인합니다.
2. **동기화 폴더 확인:** 내 컴퓨터에서 클라우드와 동기화되는 폴더의 위치를 확인합니다. (예: `C:\Users\사용자명\Google Drive` 또는 `내 PC\OneDrive`)
3. **프로그램에 폴더 설정:**
   - 프로그램 하단의 **[동기화 폴더 설정]** 버튼을 클릭합니다.
   - 폴더 선택 창이 나타나면, 위에서 확인한 Google Drive 또는 OneDrive 동기화 폴더를 선택하고 '폴더 선택' 버튼을 누릅니다.
   - "동기화 폴더가 설정되었습니다." 라는 메시지가 나타나면 성공입니다.

### 자동 동기화 사용
- 예약을 추가, 수정, 삭제할 때마다 지정된 동기화 폴더에 `schedule.ics` 파일이 자동으로 생성되거나 업데이트됩니다.
- 컴퓨터에 설치된 클라우드 데스크톱 앱이 이 파일을 감지하여 자동으로 클라우드에 업로드합니다.

### 스마트폰에서 일정 확인하기
1. 스마트폰에서 사용하시는 Google Drive 또는 OneDrive 앱을 엽니다.
2. 설정한 동기화 폴더로 이동하여 `schedule.ics` 파일을 찾습니다.
3. `schedule.ics` 파일을 누릅니다.
4. "캘린더에 추가" 또는 "캘린더로 열기"와 같은 메뉴가 나타나면, 사용하시는 스마트폰의 기본 캘린더 앱을 선택합니다.
5. 모든 예약 일정이 캘린더에 추가되거나 업데이트됩니다.

> **참고:** 이 기능은 단방향 동기화입니다. 스마트폰 캘린더에서 일정을 수정해도 데스크톱 프로그램에는 반영되지 않습니다. 일정 변경은 항상 데스크톱 프로그램에서 진행해 주세요.
